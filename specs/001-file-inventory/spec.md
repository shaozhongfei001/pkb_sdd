# Spec: 文件盘点与资产登记

## 1. 背景

本功能属于个人历史项目文档知识库 V1.1 的一部分。系统需要在保护原始文件的前提下，完成文档资产治理、解析、质量检查、证据链和项目化梳理。

## 2. 目标

实现 `文件盘点与资产登记` 能力，并与 V1.1 详细设计保持一致。

## 3. 范围

包含：

- 与本 Spec 直接相关的服务、模型、CLI/API；
- MySQL 元数据记录；
- 日志和异常处理；
- 幂等性设计；
- 对应测试用例。

## 4. 不包含

- 源代码资产分析；
- 自动删除原始文件；
- 未经确认移动原始文件；
- 上传私有文件到外部云服务。

## 5. 业务规则

1. 原始文件只读。
2. 任务必须幂等。
3. 失败必须可重试。
4. 关键结果必须有日志。
5. 涉及知识结论时必须能回溯到证据。
